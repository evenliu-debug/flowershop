# Even Shop (foreign-trade storefront) Implementation Plan

> **For agentic workers:** Optional follow-up uses Superpowers subagent-driven-development for incremental features (tests, webhooks, inventory decrement).

**Goal:** Deliver a bilingual Next.js 14 storefront with Supabase catalog/auth, Stripe + PayPal test flows, `/admin` CMS-style CRUD, and Vercel deployment.

**Architecture:** App Router with `[locale]` prefix (`en`/`zh`) via `next-intl`; Supabase RLS for customers vs admins; cart in Zustand + `localStorage`; orders written via Server Actions; Stripe Checkout Session after DB order insert; PayPal REST create/capture via Route Handlers after DB order exists.

**Tech Stack:** Next.js 14, TypeScript, Tailwind CSS, Supabase (Auth + DB + Storage), Stripe, PayPal JS SDK + REST, Zustand.
